# book-tracker

Book Tracker is a web application that allows users to keep track of the books they have read, are currently reading, have dropped, or plan to read.

## Features (Version 1)
- User registration
- User authentication
- Search books using an external API
- Personal library
- Reading status:
  - Want to Read
  - Reading
  - Read
  - Dropped

## Technologies
- Python
- FastAPI
- MySQL
- SQLAlchemy
- Git
- GitHub
